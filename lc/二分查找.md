## 69

二分法，从 1 ~ n/2 再继续二分。

**注意牛顿法的使用**

**注意下结束条件**

几个核心点在于：
1. `i < j` 还是 `i <= j`？
    - `i < j` 最终 i 和 j 相等，此时，`i`、`j` 是因为比结果 `n` 大还是小，导致的相等无从判断；也有可能导致某一个值没判断到；
2. 最终取 i、j、还是 mid 的值？
    - 如果基于 `i <= j`，i 的值（某种操作）一定是大于 n 的，j 的值（基于某种操作）一定是小于 n 的。
        - 仔细看代码可知这一点，这个非常重要。（代码逻辑上判断）
        - 当然也可以通过 `i > j` 的事实，一定要返回一个值（更粗的代码逻辑判断），题目要求返回下界，因此必然是 `j`

```
def test(n):
    i = 1
    j = n
    while i <= j:
        mid = (i+j)//2
        if mid * mid > n:
            j = mid - 1
        elif mid * mid < n:
            i = mid + 1
        else:
            return mid
    
    return j
```

因此，这道题，一定要认真对待。
